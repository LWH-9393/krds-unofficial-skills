# Public-Service Copy Patterns

Use these patterns when the screen sounds too generic or too promotional.

## Titles

Prefer:
- 장비 교육 신청
- 연구지원 공지사항
- 서비스 이용 현황
- 민원 처리 결과 조회

Avoid:
- 스마트 포털
- 혁신 플랫폼
- 디지털 서비스 허브

## Short Intro Text

Prefer:
- 신청 대상과 제출 서류를 확인한 뒤 서비스를 이용해 주세요.
- 최신 공지와 일정, 신청 현황을 한곳에서 확인할 수 있습니다.
- 처리 결과는 접수번호 기준으로 조회할 수 있습니다.

Avoid:
- 더 쉽고 빠른 경험을 제공합니다.
- 혁신적인 서비스 여정을 시작해 보세요.

## Metadata Labels

Prefer labels like:
- 작성일
- 조회
- 첨부
- 담당 부서
- 문의처
- 최종 수정일
- 처리 상태
- 접수번호

## Attachment and Contact Copy

Useful patterns:
- 첨부파일
- 다운로드
- 신청 서식
- 관련 지침
- 담당 부서
- 전화 문의
- 전자우편 문의
- 자주 묻는 질문

## Next-Step Copy

Useful patterns:
- 접수 후 담당 부서 검토가 진행됩니다.
- 처리 결과는 마이페이지와 문자로 안내합니다.
- 추가 보완이 필요할 경우 등록한 연락처로 안내합니다.
- 자세한 사항은 담당 부서로 문의해 주세요.
